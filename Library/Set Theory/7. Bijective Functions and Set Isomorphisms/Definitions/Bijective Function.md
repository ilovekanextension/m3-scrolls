**Definition.** Let $A$ and $B$ be sets. A function $f:A\to B$ is called **bijective** if $f$ is both injective and surjective.

If $f:A\to B$ is bijective, we say that (**as sets**) $A$ is **isomorphic** to $B$ and write $$A\cong B.$$
