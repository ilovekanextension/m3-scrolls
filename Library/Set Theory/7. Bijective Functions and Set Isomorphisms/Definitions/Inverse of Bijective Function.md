**Definition.** Let $A$ and $B$ be sets. Let $f:A\to B$ be bijective. By [[Relation Inverses of Bijective Functions as Functions]], $f^{-1}$ is a function from $B$ to $A$. We call this function as the **inverse function** of $f$.
